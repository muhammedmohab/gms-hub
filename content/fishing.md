# 🎣 Grandma’s Fishing Recipe 🍪

Come here dear…  
Fishing is not just relaxing… it’s a **strategy game in disguise** 😏

If you throw your hook like a fool, you’ll catch junk.  
If you follow Grandma’s recipe… you’ll feast 💛

---

# 🧠 How Fishing Works

- You drop your hook → go as deep as possible  
- Then collect fish on the way up  
- More depth = better rewards  

> [!TIP]
> The deeper you go… the sweeter the catch 🍪

---

# 🍪 Grandma’s Golden Rule

👉 Depth first… catching second  

If you don’t reach deep waters, you already lost.

---

# ⚙️ Fishing Gear (VERY IMPORTANT)

Upgrade these using fishing points:

## 🎯 Priority Order:

1. **Line** → More depth  
2. **Hook** → More capacity  
3. **Sinker** → Faster start  

> [!TIP]
> Line + Hook first = biggest boost :contentReference[oaicite:0]{index=0}

# 🎯 Core Strategy

## ⬇️ Going Down (MOST IMPORTANT)

- Avoid ALL fish  
- Stay on sides (left/right)  
- Go as deep as possible  

👉 Max depth ≈ 550m = best rewards :contentReference[oaicite:1]{index=1}  

---

## ⬆️ Going Up

Now we collect 😏

### 🎯 Priority:
1. ✨ Sparkling fish (BEST points)  
2. 🎁 Treasure chests  
3. 🧜 Mermaids  
4. 🐟 Normal fish  

> [!TIP]
> Don’t grab everything → your hook has limited space :contentReference[oaicite:2]{index=2}

---

# 🧠 Advanced Tricks (Smart Cookies Only 🍪)

## 🎯 Stick to the Sides
- Reduces collisions  
- Helps reach max depth  

---

## 🔄 Reset Bad Runs

If run is bad:
- Pause → Retreat → Retry  

👉 You keep your attempt and try again  

---

## 👀 Scout First

- Do a test run  
- Memorize fish positions  
- Retry and collect only valuable ones  

---

## 🎯 Learn Fish Value

- More stars = more points  
- Sparkling = BEST  

---
